# PersonaSurveyRepo

This repository collects and organizes research on AI coding agents and personas. Its focus is to understand which personas are useful for real development work, how they fit specific project contexts, and how persona design differs across platforms.

The current contents mainly center on React project scenarios and cross-platform persona comparison.

Personas from  https://github.com/nyldn/claude-octopus/tree/main/agents/personas

## Repository Purpose

The purpose of this repo is to turn scattered persona research, role suggestions, and platform observations into a small knowledge base that is easy to read, compare, and extend.

This repo is meant to help answer practical questions such as:

- Which personas should be prioritized for a React project?
- What kind of work is each persona best suited for?
- How do ChatGPT, Claude, and Gemini differ in persona design?
- How can persona research be used later for routing, orchestration, and review workflows?

## File Guide

### `react_project_persona_recommendations.md`

This document focuses on React projects and shares persona recommendations for frontend-oriented development work. It includes:

- Common React project tasks such as UI components, state management, hooks, API integration, performance, testing, and code review
- Recommended personas and their priority levels
- The strengths and likely responsibilities of each persona
- Why these personas are especially valuable in React project workflows
- Guidance on which personas are worth setting up first

In short, this file explains which AI personas are most useful for supporting a React project and what each of them can contribute.

### `claude_octopus_personas_combined_review.md`

This document is a broader comparison and review of persona design across platforms. It includes:

- Differences in persona or agent design between ChatGPT, Claude, and Gemini
- Observations about persona structure, description style, and extensibility
- Persona types that appear broadly useful across multiple platforms
- Candidates for a core persona set
- Direction for future orchestration, routing, and workflow design

In short, this file explains how different AI platforms approach persona design and which ideas are worth consolidating or adopting.
